Of course, we also need an LCIA method, but the definition for this is much simpler:

In [6]:
lcia_data = {
    ("simple-b", "I"): 0.1,
    ("simple-b", "II"): 0.2,
    ("simple-b", "III"): 0.3,
    ("simple-b", "IV"): 0.4
    }
lcia_method = Method(("simple", "method"))
# lcia_method.register(unit="points")
lcia_method.write(lcia_data)
lcia_method.process()

Our "final product" activity is E, and for this example, let's take 2.72 units of E. We have to reference it by its full identifier:

In [7]:
lca = LCA(demand={("simple-t", "E"): 2.72}, method=("simple", "method"))
lca.lci()  # Calculate inventory
lca.lcia()  # Calculate impact assessment
print lca.score

4.85205811378

This isn't all that helpful, as this is just a made-up example. Let's make an online report so we can learn a little more:

In [10]:
from bw2analyzer import SerializedLCAReport
report = SerializedLCAReport({("simple-t", "E"): 2.72}, method=("simple", "method"))
report.calculate()
print report.upload()
